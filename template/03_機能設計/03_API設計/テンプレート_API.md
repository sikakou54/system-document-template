# 4. 処理フロー

この API の基本フローをフローチャートで定義する。

```mermaid
flowchart TD
  Start([開始])
  P1[["(1) XXX処理"]]
  D1{"(2) XXX判定"}
  P3[["(3) XXX処理"]]
  P4["(4) 処理結果"]
  End([終了])

  Start --> P1 --> D1
  D1 -- "Yes" --> P3 --> P4 --> End
  D1 -- "No" --> End
```
