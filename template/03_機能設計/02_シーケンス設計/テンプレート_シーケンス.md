# 3. シーケンス

## 3.1 正常系シーケンス

```mermaid
sequenceDiagram
    actor U as 一般ユーザー
    participant UI as SCR-XXX 対象画面
    participant API as API-XXX 対象API
    participant MOD as MOD-XXX 対象モジュール
    participant DB as データベース

    U->>UI: 操作する
    UI->>API: 処理を要求する
    API->>MOD: 業務処理を依頼する
    MOD->>DB: データを参照または更新する（対象データモデル名）
    DB-->>MOD: 処理結果を返す
    MOD-->>API: 処理結果を返す
    API-->>UI: 処理結果を返す
    UI-->>U: 結果を表示する
```

## 3.2 代替系シーケンス

```mermaid
sequenceDiagram
    participant A as 呼び出し元
    participant B as 呼び出し先
    alt 代替条件
        A->>B: 代替処理を依頼する
        B-->>A: 代替結果を返す
    end
```

## 3.3 例外系シーケンス

```mermaid
sequenceDiagram
    participant A as 呼び出し元
    participant B as 呼び出し先
    alt 例外条件
        B-->>A: エラーを返す
    end
```
