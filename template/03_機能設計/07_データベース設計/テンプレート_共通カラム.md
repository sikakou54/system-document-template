# 2. 共通カラム一覧

| カラム名 | 型 | NULL | Key | Default | 説明 |
|---|---|---|---|---|---|
| CREATED_AT | {日時型} | No | - | {現在日時を返す既定値式} | 作成日時({保存する時刻形式・タイムゾーン}) |
| UPDATED_AT | {日時型} | No | - | {現在日時を返す既定値式} | 更新日時({保存する時刻形式・タイムゾーン})。更新時にアプリケーションで同形式の現在時刻を設定する |
| DELETED_AT | {日時型} | Yes | - | NULL | 論理削除。NULL=有効 / 値あり=削除日時({保存する時刻形式・タイムゾーン}) |

- 日時の比較可能性を保つため、{固定する保存形式(例: YYYY-MM-DDTHH:mm:ss.SSSZ のUTC固定形式)}だけを保存する。DB基盤が返す別形式(例: 空白区切り)と混在させない。
