# 4. 処理フロー

このジョブの基本フローをフローチャートで定義する。

```mermaid
flowchart TD
  Start([開始])
  L{"(1) 多重起動チェック"}
  P1["(2) XXX抽出"]
  D1{"(3) 対象有無判定"}
  P2["(4) XXX処理"]
  P3["(5) XXX更新"]
  P4["(6) レスポンス"]
  End([終了])

  Start --> L
  L -- "実行中" --> P4 --> End
  L -- "未実行" --> P1 --> D1
  D1 -- "対象あり" --> P2 --> P3 --> P4
  D1 -- "対象なし" --> P4
```
