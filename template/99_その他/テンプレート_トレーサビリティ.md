# 3. 更新ルール

| 更新契機 | 更新内容 |
|---|---|
| FR/CFR/UCの追加・変更 | 対象RQ・FR/CFR・完全修飾UCとSEQの作成要否を更新する |
| SEQの作成・変更・省略 | SEQ ID、下位設計ID、作成要否・省略理由・状態を更新する |
| 下位設計IDの予約・作成・変更 | SEQ→各設計の該当列と状態を更新する |
| ID廃止 | 行を残し、状態を「廃止」にする |

- 本対応表は関連文書の作成・更新と同一作業内で更新する(正本: docs/CLAUDE.md §6・§10-7)。
