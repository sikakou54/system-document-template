# 社員名簿管理システム 設計書（設計種別別）

架空の「社員名簿管理システム」の要求定義から、詳細レベルの画面・API・DB・モジュール・JOB・クエリ設計、トレーサビリティまでを設計種別ごとの文書に分割したもの。本番バックエンドはCloudflare Workers Paid、データベースはCloudflare D1（SQLite互換）、定期処理はCloudflare Cron TriggersからCloudflare Queuesへ分割投入する構成とし、D1 Binding・SQLへアクセスできる主体をM-006 データアクセスだけに限定する。同一内容の単一文書版は [../社員名簿管理システム_統合設計書.md](../社員名簿管理システム_統合設計書.md)。様式(テンプレート)は [../テンプレート/](../テンプレート/) を参照する。
