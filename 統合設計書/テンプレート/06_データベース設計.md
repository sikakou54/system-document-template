## 6.1 論理データモデル

```mermaid
erDiagram
    ENTITY_MAIN ||--o{ ENTITY_HISTORY : "{関連の意味}"
    ENTITY_MASTER_A ||--o{ ENTITY_HISTORY : "{関連の意味}"
    ENTITY_MAIN ||--o| ENTITY_ACCOUNT : "{関連の意味}"

    ENTITY_MAIN {
        string entity_id PK
        string entity_code UK
        string status
    }
    ENTITY_HISTORY {
        string history_id PK
        string entity_id FK
        string master_id FK
        date effective_from
        date effective_to
    }
    ENTITY_MASTER_A {
        string master_id PK
        string master_code UK
        boolean active
    }
    ENTITY_ACCOUNT {
        string account_id PK
        string entity_id FK
        boolean active
    }
```
