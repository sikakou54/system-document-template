## 1. 基本情報

| 項目 | 内容 |
|---|---|
| 要件ID | BR-008 |
| 要件名 | 利用量に応じて公平に課金する |
| 概要 | 有料会議室の利用時間に応じて従量で課金し、月次でまとめて請求できるようにする |
| 実現したい状態 | 有料会議室の利用時間に応じ従量で課金し、月次で請求する |
| 背景 | 有料会議室の費用を利用量に関係なく一律で負担しており、利用実態と負担が見合っていない |

## 2. 業務要件

この業務要件で実現したいことを定義する。

| No | 実現したいこと |
|---|---|
| 1 | 有料(利用単価が設定された)会議室の利用時間に応じて課金できるようにする(従量課金) |
| 2 | 利用者が有料会議室を利用する前に支払い方法を登録できるようにする |
| 3 | 利用が完了した分の利用量を計上し、月次でまとめて請求できるようにする |
| 4 | 利用者が自分の当月の利用量・請求見込みと請求履歴を確認できるようにする |

## 3. 成功指標・期待効果

この業務要件の達成度を測る指標と期待効果を定義する。

| No | 区分 | 内容 |
|---|---|---|
| 1 | 成功指標 | 有料会議室の利用に対し、利用時間に応じた金額が月次で請求される |
| 2 | 期待効果 | 利用量に応じた公平な費用負担が実現する |
| 3 | 期待効果 | 有料会議室の利用状況とコストが対応づき、費用の透明性が高まる |

## 4. 関連(実現する機能要件)

この業務要件を実現する機能要件(FR)・共通機能要件(CFR)を定義する。

| ID | 名称 | 実現内容 |
|---|---|---|
| FR-008 | 会議室利用の従量課金 | 有料会議室の利用時間に応じて従量課金し、月次で請求する。利用者は支払い方法を登録する |

## 5. 前提・制約

この業務要件の前提と制約を定義する。

| No | 区分 | 内容 |
|---|---|---|
| 1 | 前提 | 会議室に利用単価が設定されている(利用単価0は無料として課金対象外) |
| 2 | 前提 | 利用者が有料会議室を利用する前に支払い方法を登録している |
| 3 | 制約 | 課金対象は利用が完了した有料会議室の予約に限る |
| 4 | 制約 | 課金・請求の具体的な実現方式(決済サービス等)は本業務要件では扱わない(FR-008 で扱う範囲を定義) |
