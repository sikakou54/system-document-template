# 1. 基本情報

| 項目 | 内容 |
|---|---|
| 要件ID | BR-001 |
| 要件名 | 空き会議室を見つけやすくする |
| 概要 | 日時・人数・設備といった条件から、空いている会議室を短時間で見つけられるようにする |
| 実現したい状態 | 利用者が日時・人数・設備から空き会議室を短時間で見つけられる |
| 背景 | 会議室の空き状況が可視化されておらず、空きを探すのに手間と時間がかかっている |

# 2. 業務要件

この業務要件で実現したいことを定義する。

| No | 実現したいこと |
|---|---|
| 1 | 利用したい日時に空いている会議室を絞り込めるようにする |
| 2 | 人数や必要な設備に合う会議室だけを候補として提示できるようにする |
| 3 | 条件に合う会議室が無い場合はその旨がわかるようにする |

# 3. 成功指標・期待効果

この業務要件の達成度を測る指標と期待効果を定義する。

| No | 区分 | 内容 |
|---|---|---|
| 1 | 成功指標 | 条件を指定してから空き会議室の候補が提示されるまでの操作が数ステップで完了する |
| 2 | 期待効果 | 会議室探しにかかる時間が短縮され、会議準備の負担が減る |
| 3 | 期待効果 | 空き会議室の見落としが減り、会議室が有効に活用される |

# 4. 関連(実現する機能要件)

この業務要件を実現する機能要件(FR)・共通機能要件(CFR)を定義する。

| ID | 名称 | 実現内容 |
|---|---|---|
| FR-001 | 会議室検索 | 日時・人数・設備を条件に空き会議室を検索し一覧提示する |

# 5. 前提・制約

この業務要件の前提と制約を定義する。

| No | 区分 | 内容 |
|---|---|---|
| 1 | 前提 | 会議室・設備・既存予約の情報が登録されている |
| 2 | 制約 | 会議室の予約登録はこの業務要件では扱わない(BR-002) |
