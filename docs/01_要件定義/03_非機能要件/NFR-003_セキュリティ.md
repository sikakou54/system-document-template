# 1. 基本情報

| 項目 | 内容 |
|---|---|
| 要件ID | NFR-003 |
| 分類 | セキュリティ |
| 対象範囲 | 全機能 |
| 概要 | パスワードは平文保存せずハッシュ化して保存する |

# 2. 品質要求・達成基準・検証方法

この非機能要件の品質要求と、その達成基準・検証方法を定義する。

| 品質要求 | 達成基準 | 検証方法 |
|---|---|---|
| パスワードは平文保存せずハッシュ化して保存する | 安全なハッシュ方式を適用し、平文パスワードを保存しない(ハッシュ方式の詳細な実装は MOD-008 で定義する) | コードレビューでハッシュ方式を確認し、保存データ確認で平文が保存されていないことを確認する |
