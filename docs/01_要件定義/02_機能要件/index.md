## 1. 概要

MeetRoom の要件定義の総括。個別の機能要件(FR)の一覧を定義する。
システム全体の要件、機能横断の非機能要件(NFR)、共通機能要件(CFR)、共通業務ルール(RULE-COM)は CFR-000_システム共通要件.md に集約する（本一覧には再記載しない）。
各機能要件の詳細は本ディレクトリの FR-XXX_要件名.md に記載する。

## 2. 共通要件（CFR-000 参照）

システム全体の要件、機能横断の非機能要件、共通機能要件、共通業務ルールは CFR-000_システム共通要件.md で定義する。
機能固有の非機能要件は各 FR の「機能固有の非機能要件」節に記載する。

## 3. 個別機能要件一覧

| ID | 名称 | 概要 |
|---|---|---|
| FR-001 | 会議室検索 | 日時・人数・設備を条件に空き会議室を検索できる |
| FR-002 | 会議室予約 | 空き会議室を予約できる。同一会議室・同一時間帯の二重予約は不可 |
| FR-003 | 予約変更・キャンセル | 自分の予約を変更・キャンセルできる |
| FR-004 | 予約リマインド通知 | 予約開始前に予約者へメールでリマインド通知する |
| FR-005 | 会議室・設備管理 | 管理者が会議室と設備を登録・編集できる |
| FR-006 | 利用実績レポート | 管理者が会議室ごとの月次利用実績を確認できる |
| FR-007 | ユーザー認証（廃止） | 認証・認可は共通機能要件へ移行(CFR-000 の CFR-001 認証 / CFR-002 認可・権限制御)。本IDは欠番とし再利用しない |
| FR-008 | 会議室利用の従量課金 | 有料会議室の利用時間に応じて従量課金し、月次で請求する。利用者は事前に支払い方法を登録する |
