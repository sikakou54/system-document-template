# 1. 基本情報

| 項目 | 内容 |
|---|---|
| クエリID | SQL-025 |
| クエリ名 | 予約登録 |
| 概要 | 予約を1件登録し(予約済・リマインド未送信で作成)、登録した予約を返す。 |
| 用途 | 予約の登録(ReservationService)。 |
| 使用元 | MOD-003 |

# 2. 対象テーブル

| テーブル | 物理名 | 用途 |
|---|---|---|
| TBL-003 | T_RESERVATIONS | 登録対象(予約の INSERT) |

# 3. 入力パラメータ

| パラメータ | 論理名 | 型 | 必須 | 説明・制約 |
|---|---|---|---|---|
| :room_id | 会議室ID | INTEGER | Yes | 予約対象の会議室ID |
| :user_id | ユーザーID | INTEGER | Yes | 予約する利用者の ID |
| :title | 予約タイトル | TEXT | Yes | 予約タイトル |
| :start_at | 利用開始日時 | TEXT | Yes | ISO8601(UTC)。:start_at ＜ :end_at |
| :end_at | 利用終了日時 | TEXT | Yes | ISO8601(UTC) |

# 4. クエリ

予約を INSERT する。STATUS=1(予約済)、REMIND_STATUS=1(未送信)で作成する。登録した予約を返す。

```sql
INSERT INTO T_RESERVATIONS (
  ROOM_ID, USER_ID, TITLE, START_AT, END_AT, STATUS, REMIND_STATUS
) VALUES (
  :room_id, :user_id, :title, :start_at, :end_at, 1, 1   -- STATUS=1:予約済(ENM-1) / REMIND_STATUS=1:未送信(ENM-2)
)
RETURNING ID, ROOM_ID, USER_ID, TITLE, START_AT, END_AT, STATUS, REMIND_STATUS;
```

# 5. 出力(列)

| 列名 | 論理名 | 型 | 説明 |
|---|---|---|---|
| ID | 予約ID | INTEGER | 採番された予約の ID |
| ROOM_ID | 会議室ID | INTEGER | 予約対象の会議室ID |
| USER_ID | ユーザーID | INTEGER | 予約者の ID |
| TITLE | 予約タイトル | TEXT | 予約タイトル |
| START_AT | 利用開始日時 | TEXT | ISO8601(UTC) |
| END_AT | 利用終了日時 | TEXT | ISO8601(UTC) |
| STATUS | 予約ステータス | INTEGER | 1=予約済(TBL-003/ENM-1) |
| REMIND_STATUS | リマインド通知ステータス | INTEGER | 1=未送信(TBL-003/ENM-2) |

# 6. 補足(性能・インデックス)

| 項目 | 内容 |
|---|---|
| 利用インデックス | なし(INSERT) |
| 想定件数 | 1件 |
