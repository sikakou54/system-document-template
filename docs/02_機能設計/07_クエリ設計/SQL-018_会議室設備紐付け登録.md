# 1. 基本情報

| 項目 | 内容 |
|---|---|
| クエリID | SQL-018 |
| クエリ名 | 会議室設備紐付け登録 |
| 概要 | 会議室と設備の紐付けを1件登録する。 |
| 用途 | 会議室登録・編集時の設備紐付けの登録・付け替え(RoomManagementService)。 |
| 使用元 | MOD-004 |

# 2. 対象テーブル

| テーブル | 物理名 | 用途 |
|---|---|---|
| TBL-005 | M_ROOM_EQUIPMENTS | 登録対象(会議室×設備の紐付け INSERT) |

# 3. 入力パラメータ

| パラメータ | 論理名 | 型 | 必須 | 説明・制約 |
|---|---|---|---|---|
| :room_id | 会議室ID | INTEGER | Yes | 紐付ける会議室の ID |
| :equipment_id | 設備ID | INTEGER | Yes | 紐付ける設備の ID |

# 4. クエリ

会議室と設備の紐付けを INSERT する。紐付ける設備ごとに実行する。

```sql
INSERT INTO M_ROOM_EQUIPMENTS (
  ROOM_ID, EQUIPMENT_ID
) VALUES (
  :room_id, :equipment_id
)
RETURNING ID, ROOM_ID, EQUIPMENT_ID;
```

# 5. 出力(列)

| 列名 | 論理名 | 型 | 説明 |
|---|---|---|---|
| ID | 紐付けID | INTEGER | 採番された紐付けの ID |
| ROOM_ID | 会議室ID | INTEGER | 会議室の ID |
| EQUIPMENT_ID | 設備ID | INTEGER | 設備の ID |

# 6. 補足(性能・インデックス)

| 項目 | 内容 |
|---|---|
| 利用インデックス | UX_ROOM_EQUIPMENTS_ROOM_EQUIP(ROOM_ID, EQUIPMENT_ID 一意制約) |
| 想定件数 | 紐付ける設備の件数分(1件ずつ実行) |
| 注意点 | 同一の会議室×設備の組み合わせは UX_ROOM_EQUIPMENTS_ROOM_EQUIP により重複登録できない。編集時の付け替えは SQL-019(既存削除)後に本クエリで登録し直す |
