# 1. 基本情報

| 項目 | 内容 |
|---|---|
| クエリID | SQL-023 |
| クエリ名 | 自予約一覧取得 |
| 概要 | 利用者の予約を、予約ステータス・利用開始日の期間で絞り込み、会議室名付き・利用開始日時昇順で取得する。 |
| 用途 | マイ予約一覧の取得。 |
| 使用元 | MOD-003 |

# 2. 対象テーブル

| テーブル | 物理名 | 用途 |
|---|---|---|
| TBL-003 | T_RESERVATIONS | 抽出対象(利用者の予約) |
| TBL-002 | M_ROOMS | 会議室名(NAME)の取得 |

# 3. 入力パラメータ

| パラメータ | 論理名 | 型 | 必須 | 説明・制約 |
|---|---|---|---|---|
| :user_id | ユーザーID | INTEGER | Yes | 一覧を取得する利用者の ID |
| :status | 予約ステータス | INTEGER | No | 絞り込む予約ステータス(DEF-001/CODE-004)。NULL のとき絞り込まない |
| :from | 期間開始 | TEXT | No | 利用開始日時の下限(ISO8601/UTC)。NULL のとき下限なし |
| :to | 期間終了 | TEXT | No | 利用開始日時の上限(ISO8601/UTC)。NULL のとき上限なし |

# 4. クエリ

利用者(USER_ID)の有効な予約を、任意条件(ステータス・利用開始日時の期間)で絞り込み、ROOM_ID で M_ROOMS に結合して会議室名付きで取得する。任意条件は NULL のとき無効化する。利用開始日時昇順で並べる。

```sql
SELECT
  r.ID            AS ID,
  r.ROOM_ID       AS ROOM_ID,
  m.NAME          AS ROOM_NAME,
  r.USER_ID       AS USER_ID,
  r.TITLE         AS TITLE,
  r.START_AT      AS START_AT,
  r.END_AT        AS END_AT,
  r.STATUS        AS STATUS,
  r.REMIND_STATUS AS REMIND_STATUS
FROM T_RESERVATIONS r
INNER JOIN M_ROOMS m
  ON m.ID = r.ROOM_ID
WHERE r.USER_ID = :user_id
  AND r.DELETED_AT IS NULL
  AND (:status IS NULL OR r.STATUS = :status)          -- 予約ステータス(DEF-001/CODE-004)
  AND (:from   IS NULL OR r.START_AT >= :from)
  AND (:to     IS NULL OR r.START_AT <= :to)
ORDER BY r.START_AT ASC, r.ID ASC;
```

# 5. 出力(列)

| 列名 | 論理名 | 型 | 説明 |
|---|---|---|---|
| ID | 予約ID | INTEGER | 予約の ID |
| ROOM_ID | 会議室ID | INTEGER | 予約対象の会議室ID |
| ROOM_NAME | 会議室名 | TEXT | 会議室名 |
| USER_ID | ユーザーID | INTEGER | 予約者の ID |
| TITLE | 予約タイトル | TEXT | 予約タイトル |
| START_AT | 利用開始日時 | TEXT | ISO8601(UTC) |
| END_AT | 利用終了日時 | TEXT | ISO8601(UTC) |
| STATUS | 予約ステータス | INTEGER | DEF-001/CODE-004 |
| REMIND_STATUS | リマインド通知ステータス | INTEGER | DEF-001/CODE-005 |
