# 1. 基本情報

| 項目 | 内容 |
|---|---|
| クエリID | SQL-012 |
| クエリ名 | 会議室更新 |
| 概要 | 会議室1件の属性(名称・収容人数・設置場所・利用単価・ステータス・備考)を更新し、更新後の会議室を返す。 |
| 用途 | 会議室の編集(RoomManagementService)。 |
| 使用元 | MOD-004 |

# 2. 対象テーブル

| テーブル | 物理名 | 用途 |
|---|---|---|
| TBL-002 | M_ROOMS | 更新対象(会議室属性の UPDATE) |

# 3. 入力パラメータ

| パラメータ | 論理名 | 型 | 必須 | 説明・制約 |
|---|---|---|---|---|
| :id | 会議室ID | INTEGER | Yes | 更新対象会議室の ID |
| :name | 会議室名 | TEXT | Yes | 会議室名 |
| :capacity | 収容人数 | INTEGER | Yes | 1以上 |
| :location | 設置場所 | TEXT | Yes | 設置場所 |
| :hourly_rate | 利用単価 | INTEGER | Yes | 0以上(0=無料) |
| :status | 会議室ステータス | INTEGER | Yes | DEF-001/CODE-003 |
| :note | 備考 | TEXT | No | 備考(NULL可) |
| :updated_at | 更新日時 | TEXT | Yes | ISO8601(UTC)。アプリで現在時刻を設定(共通カラム.md) |

# 4. クエリ

有効(DELETED_AT IS NULL)な会議室の属性を更新する。更新後の会議室を返す。

```sql
UPDATE M_ROOMS
SET NAME        = :name,
    CAPACITY    = :capacity,
    LOCATION    = :location,
    HOURLY_RATE = :hourly_rate,
    STATUS      = :status,                             -- DEF-001/CODE-003
    NOTE        = :note,
    UPDATED_AT  = :updated_at
WHERE ID = :id
  AND DELETED_AT IS NULL
RETURNING ID, NAME, CAPACITY, LOCATION, HOURLY_RATE, STATUS, NOTE;
```

# 5. 出力(列)

| 列名 | 論理名 | 型 | 説明 |
|---|---|---|---|
| ID | 会議室ID | INTEGER | 会議室の ID |
| NAME | 会議室名 | TEXT | 会議室名 |
| CAPACITY | 収容人数 | INTEGER | 収容人数 |
| LOCATION | 設置場所 | TEXT | 設置場所 |
| HOURLY_RATE | 利用単価 | INTEGER | 1時間あたり利用単価(円)。0=無料 |
| STATUS | 会議室ステータス | INTEGER | DEF-001/CODE-003 |
| NOTE | 備考 | TEXT | 備考(NULL可) |

# 6. 補足(性能・インデックス)

|---|---|
