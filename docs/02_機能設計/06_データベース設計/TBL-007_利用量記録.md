# 1. 基本情報

| 項目 | 内容 |
|---|---|
| テーブルID | TBL-007 |
| 物理名 | T_USAGE_RECORDS |
| 論理名 | 利用量記録トランザクション |
| 概要 | 有料会議室の完了予約ごとに利用量(時間)を記録し、Stripe Meter Event として計上する従量課金用トランザクション(FR-008)。 |
