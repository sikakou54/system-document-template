## 1. 基本情報

| 項目 | 内容 |
|---|---|
| テーブルID | TBL-008 |
| 物理名 | T_INVOICES |
| 論理名 | 請求トランザクション |
| 概要 | Stripe が発行する従量課金の月次請求(invoice)を保持するトランザクション。Webhook(invoice.paid 等)で状態を更新する(FR-008)。 |

## 5. 列挙値

### ENM-1 請求ステータス

| 値 | 表示名 | 説明 |
|---|---|---|
| 1 | 確定前 | Stripe 上で draft(集計中・未確定) |
| 2 | 請求済 | Stripe 上で open(請求書発行済・未払い) |
| 3 | 支払済 | Stripe 上で paid(支払い完了) |
| 4 | 失敗 | 支払い失敗(invoice.payment_failed) |
