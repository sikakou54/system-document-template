## 1. 基本情報

| 項目 | 内容 |
|---|---|
| 画面ID | SCR-002 |
| 画面名 | 会議室検索 |
| 概要 | 日時・人数・設備を条件に空き会議室を検索し、予約する会議室を選択する画面 |
| トレース元 | UC-001, UC-002 |
| URL / ルート | /rooms |
| 利用可能ロール | 一般 / 管理者 |

## 2. 画面レイアウト

![会議室検索 画面レイアウト](mockups/SCR-002.png)

## 3. 初期表示

| 項目 | 内容 |
|---|---|
| 表示時に呼び出すAPI | API-002 |
| デフォルト値 | 当日・時間指定なし・全会議室 |
| ソート順 | 会議室名昇順 |
| 0件時の表示 | 「条件に合う会議室がありません」を表示 |

## 4. 画面項目

| 項目ID | 項目名 | 種別 | 表示/入力 | 必須 | 初期値 | 備考 |
|---|---|---|---|---|---|---|
| ITM-01 | 利用日 | date | 入力 | Yes | 当日 | - |
| ITM-02 | 開始時刻 | select | 入力 | No | - | 15分刻み |
| ITM-03 | 終了時刻 | select | 入力 | No | - | 15分刻み |
| ITM-04 | 人数 | number | 入力 | No | - | - |
| ITM-05 | 設備 | select | 入力 | No | - | 選択肢は equipments から |
| ITM-06 | 検索ボタン | button | 入力 | - | - | EVT-01 を発火 |
| ITM-07 | 結果一覧 | 一覧 | 表示 | - | - | 各行に「予約する」ボタンを持つ(EVT-02) |

## 5. 画面イベント

| イベントID | イベント名 | 発火条件 | 呼び出しAPI | 成功時 | 失敗時 |
|---|---|---|---|---|---|
| EVT-01 | 検索 | 検索ボタン押下 | API-002 | 検索結果を結果一覧に表示 | - |
| EVT-02 | 予約する | 結果行の「予約する」押下 | - | SCR-003 へ遷移(会議室・日時を引き継ぐ) | - |

## 6. 入力チェック

| 対象項目 | チェック内容 | 表示メッセージ |
|---|---|---|
| 開始時刻・終了時刻 | 開始 < 終了 であること(クライアント側) | - |

## 7. 表示制御

| 条件 | 対象 | 制御内容 |
|---|---|---|
| 検索結果が0件 | 結果一覧 | 非表示 |
| 検索結果が0件 | 0件メッセージ | 表示 |

## 8. 画面遷移

| 遷移先 | トリガ |
|---|---|
| SCR-003 | 結果行の「予約する」押下(EVT-02。会議室・日時を引き継ぐ) |

## 9. メッセージ一覧

本画面が参照する画面表示文言(MSG)を以下にインライン定義する。対応ERR は当該メッセージの表示契機となるエラー(なしは -)。

| MSG ID | 種別 | 文言 | 対応ERR |
|---|---|---|---|
| MSG-007 | 情報 | 条件に合う会議室がありません | - |
| MSG-008 | エラー | 終了時刻は開始時刻より後の時刻を指定してください。 | - |
| MSG-009 | エラー | 検索条件が正しくありません。入力内容を確認してください。 | ERR-006 |
