## 1. 概要

MeetRoom の画面(SCR-XXX)の一覧。各画面の項目・イベント・入力チェック・表示制御は各 SCR 文書に定義する。
画面表示文言(MSG-XXX)はカタログを設けず、各画面が参照する MSG を当該 SCR 文書の「メッセージ一覧」節にインライン定義する。各 SCR 文書はその節を ID で参照する。
