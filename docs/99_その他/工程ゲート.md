# 1. 目的

本ファイルは、MeetRoom の必須の設計順序(Step 1〜8)における**工程ゲートの状態・証跡・承認の正本**である。状態語彙と遷移条件は [CLAUDE.md](../CLAUDE.md) §7、必須作成順と出口条件は §2.1、ゲート運用手順は §9.3 を正本とする。

# 2. ゲート状態の語彙

文書状態(未着手/作成中/レビュー中/確定/廃止)とは別に管理する。承認は「承認済」でのみ表し、承認者はユーザーまたは設計責任者に限る(作成者は自分の成果物を「承認済」にしない)。

| 状態 | 意味 |
|---|---|
| 未着手 | 当該 Step の成果物にまだ着手していない |
| 作業中 | 成果物を作成・修正中 |
| レビュー中 | 作成者が完了宣言し、レビュー・承認を依頼済み |
| 承認済 | ユーザーまたは設計責任者が出口条件を確認し承認した(次 Step を確定してよい) |
| 失効 | 上流変更により承認が無効化された(影響分析後に再レビューする) |
| 廃止 | 当該 Step を使用しない |
