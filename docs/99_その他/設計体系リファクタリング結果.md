# 7. 変更後の正本管理ルール

正本の全一覧は [CLAUDE.md](../CLAUDE.md) §3.2を参照する。要点は次のとおり。

| 情報 | 正本 |
|---|---|
| システムが実現すること・利用者から見た振る舞い | FR/CFR・UC |
| 概念エンティティ・属性・関連 | MDL・概念ER図 |
| コンポーネント間の順序・受け渡し・分岐 | SEQ |
| 画面項目・操作・表示・ユーザー向けMSG | SCR |
| API入出力・HTTP・認可 | API/API-COM |
| 業務ロジック・条件判定 | MOD |
| テーブル・カラム・物理制約 | TBL・ER図 |
| データ操作条件・SQL | SQL |
| エラーコード・HTTPステータス・開発者向け文言 | API配下のエラーメッセージ一覧 |
| 権限要件／実装制御 | CFR-002／SCR・API |
| 外部連携方針／個別契約 | 概要設計／該当API・JOB・MOD |

ERR・MSG・権限・外部IFは、既存の正本配置を維持しつつSEQからの反映先を明文化した。独立カテゴリを新設して仕様を二重化していない。
