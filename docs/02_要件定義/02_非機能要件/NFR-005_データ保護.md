# 1. 基本情報

| 項目 | 内容 |
|---|---|
| 要件ID | NFR-005 |
| 分類 | データ保護 |
| 対象範囲 | システム全体(業務データ) |
| 概要 | 業務データを定期的にバックアップし、障害時のデータ消失に備える |
| 状態 | 確定 |

# 2. 品質要求・達成基準・検証方法

この非機能要件の品質要求と、その達成基準・検証方法を定義する。

| 品質要求 | 達成基準 | 検証方法 |
|---|---|---|
| 業務データを定期的にバックアップし、障害時に目標時間内で復旧する | RPO 15分以内、RTO 4時間以内とする。RPOを満たす世代を保持し、少なくとも日次で独立したバックアップを取得する | バックアップ取得を日次記録で確認し、四半期ごとに本番と分離した環境へ復元してデータ整合性・RPO・RTOを検証する |
