# 1. 基本情報

| 項目 | 内容 |
|---|---|
| 要件ID | NFR-004 |
| 分類 | セキュリティ(通信) |
| 対象範囲 | 全機能(利用者との通信・システムからの送信メール) |
| 概要 | 通信を暗号化し、システムが送信するメールのなりすましを防ぐ |
| 状態 | 確定 |

# 2. 品質要求・達成基準・検証方法

この非機能要件の品質要求と、その達成基準・検証方法を定義する。

| 品質要求 | 達成基準 | 検証方法 |
|---|---|---|
| 利用者との通信経路を暗号化し、システムが送信するメールのなりすましを防ぐ | 利用者との通信を HTTPS で暗号化する。本番環境からメールを送信するすべての送信元ドメインに SPF/DKIM を設定し、認証済みドメイン以外から送信しない | 通信が HTTPS であること、および本番送信元ドメイン一覧の全件で SPF/DKIM が有効であることをリリース前と設定変更時に確認する |
