# 1. 基本情報

| 項目 | 内容 |
|---|---|
| 要件ID | NFR-006 |
| 分類 | 監査 |
| 対象範囲 | 予約・課金・会議室管理などの重要操作 |
| 概要 | 重要操作の記録を保持し、後から追跡できるようにする |
| 状態 | 確定 |

# 2. 品質要求・達成基準・検証方法

この非機能要件の品質要求と、その達成基準・検証方法を定義する。

| 品質要求 | 達成基準 | 検証方法 |
|---|---|---|
| 重要操作の記録を保持し、後から追跡できるようにする | 予約・課金・会議室管理などの重要操作について、操作日時、操作者、操作種別、操作対象種別・対象ID、操作結果、相関IDを必須記録し、作成から1年間保持する。保持期間中は追記のみとし、通常の業務処理から更新・削除できないこと | 重要操作ごとの監査ログ項目充足を結合試験で確認し、アクセス制御を権限試験で、1年保持設定と期限到来後の廃棄を運用設定・定期点検で確認する |
