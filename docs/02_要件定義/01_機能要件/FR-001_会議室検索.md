# FR-001：会議室検索機能 仕様書

## 1. 基本情報
| 項目 | 内容 |
|---|---|
| 要件ID | FR-001 |
| 要件名 | 会議室検索 |
| 概要 | 日時・人数・設備を条件に空き会議室を検索できる |
| 目的 | 会議室の空き状況の不透明さを解消し、条件に合う会議室を迅速に見つけられるようにする |
| 対象ロール | 利用者 |

## 2. 業務ルール（制約事項）
システムが遵守すべき業務上の絶対ルールを定義する。このルールがユースケースのシナリオ分岐の根拠となる。

| No | ルール |
|---|---|
| 1 | 指定した時間帯に予約が入っていない会議室のみを検索結果に表示する |
| 2 | 指定人数が会議室の収容人数を超える会議室は検索結果に表示しない |
| 3 | 指定設備を持たない会議室は検索結果に表示しない |
| 4 | 開始日時が終了日時以降の場合は検索できない |
| 5 | 利用不可状態の会議室は検索結果に表示しない |
| 6 | 予約登録はこの要件では扱わない |
| 7 | 会議室情報の登録・編集・削除はこの要件では扱わない |
| 8 | 設備情報の登録・編集・削除はこの要件では扱わない |

## 3. 受入条件
この要件を満たしていることを確認する条件（テストの合否判定基準）を定義する。

| No | 条件 |
|---|---|
| 1 | 開始日時・終了日時を指定して検索すると、指定時間帯に空きがある会議室のみ表示される |
| 2 | 人数を指定して検索すると、指定人数を収容できる会議室のみ表示される |
| 3 | 設備を指定して検索すると、指定設備を持つ会議室のみ表示される |
| 4 | 条件に合致する空き会議室が存在しない場合は、結果が0件であることがわかる |
| 5 | 開始日時が終了日時以降の場合は、検索できないことがわかる |

## 4. システムユースケース
この要件を実現する振る舞いを、データなどのパターンに応じてユースケース単位で定義する。

### 4.1 【UC-01】空き会議室を検索する（一般ユーザー向け）
日時・人数・設備を指定して、その時間帯に利用できる会議室を検索する通常のフロー。

**事前条件**

| No | 条件 |
|---|---|
| 1 | ログイン済みであること |
| 2 | 会議室、設備、既存予約の情報が登録されていること |

**事後条件**

| No | 条件 |
|---|---|
| 1 | 条件に合致する空き会議室の一覧が利用者に提示されていること |
| 2 | 会議室の予約状態は変更されていないこと |

**入力データ**

| 情報 | 要否 | 内容 |
|---|---|---|
| 利用開始日時 | 必須 | 会議室を使い始める日時 |
| 利用終了日時 | 必須 | 会議室を使い終える日時 |
| 利用人数 | 任意 | 会議の参加人数。収容人数による絞り込みに用いる |
| 設備 | 任意 | 必要とする設備。設備の有無による絞り込みに用いる |

**出力データ**

| 情報 | 内容 |
|---|---|
| 会議室名 | 空き会議室の名称 |
| 収容人数 | 会議室に収容できる人数 |
| 場所 | 会議室の場所 |
| 設備 | 会議室が備えている設備 |
| 空き状況 | 指定した時間帯に利用できること |

**基本フロー**

| Step | アクター | 操作/処理 | 結果 |
|---|---|---|---|
| 1 | 利用者 | 利用開始日時・利用終了日時・人数・設備などの条件を入力して検索する | 検索条件が受け付けられる |
| 2 | システム | 条件に合致し、その時間帯に予約が入っておらず、かつ利用可能状態である会議室を絞り込む | 条件を満たす空き会議室が抽出される |
| 3 | システム | 抽出した空き会議室を一覧で提示する | 空き会議室一覧が表示される |

**代替フロー**

| ALT ID | 分岐Step | 条件 | フロー |
|---|---|---|---|
| ALT-1 | Step3 | 条件に合致する空き会議室が0件 | 0件であることを確認し、条件を変えて再検索する |

**例外フロー**

| EXC ID | 発生Step | 条件 | エラーメッセージ | 対応 |
|---|---|---|---|---|
| EXC-1 | Step1 | 利用開始日時が未指定 | 利用開始日時を入力してください。 | Step1 に戻る |
| EXC-2 | Step1 | 利用終了日時が未指定 | 利用終了日時を入力してください。 | Step1 に戻る |
| EXC-3 | Step1 | 利用開始日時が利用終了日時以降 | 利用終了日時は利用開始日時より後の日時を指定してください。 | Step1 に戻る |
