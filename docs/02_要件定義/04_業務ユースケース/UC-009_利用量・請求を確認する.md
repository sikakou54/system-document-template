# 1. 基本情報

| 項目 | 内容 |
|---|---|
| UC ID | UC-009 |
| ユースケース名 | 利用量・請求を確認する |
| 主アクター | 一般ユーザー |
| トレース元 | FR-008 |
| 概要 | 当月の利用量・請求見込みと請求履歴を確認する |

# 2. 事前条件

| No | 条件 |
|---|---|
| 1 | ログイン済み(UC-007 完了) |

# 3. 基本フロー

| Step | アクター | 操作/処理 | 結果 |
|---|---|---|---|
| 1 | 利用者 | 支払い方法・請求の画面を開く | 画面が表示される |
| 2 | 利用者 | 確認する対象月を指定する(未指定時は当月) | 表示対象が定まる |
| 3 | システム | 対象月の利用量・請求見込みと請求履歴を集計する | 利用量・請求情報が算出される |
| 4 | システム | 利用量・請求見込み・請求履歴を提示する | 利用量・請求が表示される |

# 4. 代替フロー

| ALT ID | 分岐Step | 条件 | フロー |
|---|---|---|---|
| ALT-1 | Step2 | 過去の対象月を指定する | 指定月の利用量・請求履歴を確認する |
| ALT-2 | Step4 | 対象月に有料会議室の利用がない | 利用量0・請求見込みなしであることを確認する |

# 5. 例外フロー

| EXC ID | 発生Step | 条件 | 対応 |
|---|---|---|---|
| - | - | なし | - |

# 6. 事後条件

| No | 条件 |
|---|---|
| 1 | 対象月の利用量・請求見込み・請求履歴が利用者に提示されている |
| 2 | 課金・請求データは変更されていない |
