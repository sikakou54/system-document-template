# 1. 概要

MeetRoom の利用者・システムの振る舞い(フロー)をユースケースとして定義する。各ユースケースの詳細は UC-XXX_ユースケース名.md に記載する。

# 2. 一覧

| ID | 名称 | 概要 | 主アクター | トレース元 |
|---|---|---|---|---|
| UC-001 | 空き会議室を検索する | 日時・人数・設備を条件に空き会議室を検索する | 一般ユーザー | FR-001 |
| UC-002 | 会議室を予約する | 空き会議室を検索・選択し、タイトルを入力して予約を登録する | 一般ユーザー | FR-002 |
| UC-003 | 予約を変更する | 自分の予約の内容を変更する | 一般ユーザー | FR-003 |
| UC-004 | 予約をキャンセルする | 自分の予約をキャンセルする | 一般ユーザー | FR-003 |
| UC-005 | 会議室を登録・編集する | 管理者が会議室と設備を登録・編集する | 管理者 | FR-005 |
| UC-006 | 利用実績を確認する | 管理者が会議室ごとの月次利用実績を確認する | 管理者 | FR-006 |
| UC-007 | ログインする | 登録ユーザーがログインしてシステムを利用可能な状態にする | 一般ユーザー / 管理者 | CFR-001 |
| UC-008 | 支払い方法を登録する | 支払い方法を登録し、有料会議室の従量課金を有効化する | 一般ユーザー | FR-008 |
| UC-009 | 利用量・請求を確認する | 当月の利用量・請求見込みと請求履歴を確認する | 一般ユーザー | FR-008 |
