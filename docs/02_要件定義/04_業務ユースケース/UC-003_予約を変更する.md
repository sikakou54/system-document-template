# 1. 基本情報

| 項目 | 内容 |
|---|---|
| UC ID | UC-003 |
| ユースケース名 | 予約を変更する |
| 主アクター | 一般ユーザー |
| トレース元 | FR-003 |
| 概要 | 自分の予約の内容を変更する |

# 2. 事前条件

| No | 条件 |
|---|---|
| 1 | ログイン済み(UC-007 完了) |
| 2 | 変更対象の自分の予約が存在する |

# 3. 基本フロー

| Step | アクター | 操作/処理 | 結果 |
|---|---|---|---|
| 1 | 利用者 | 自分の予約一覧を表示する | 予約一覧が表示される |
| 2 | 利用者 | 変更する予約を選択する | 変更入力に進む |
| 3 | 利用者 | 日時・タイトルを変更して確定する | 変更登録が要求される |
| 4 | システム | 予約を変更する | 予約が変更される |
| 5 | システム | 完了メッセージを表示する | 完了メッセージが表示される |

# 4. 代替フロー

| ALT ID | 分岐Step | 条件 | フロー |
|---|---|---|---|
| ALT-1 | Step3 | 変更を取りやめる | 変更せず元の予約のまま一覧に戻る |

# 5. 例外フロー

| EXC ID | 発生Step | 条件 | 対応 |
|---|---|---|---|
| EXC-1 | Step4 | 変更後の日時が他の予約と重複する | エラー表示、Step3 へ戻る |
| EXC-2 | Step4 | 変更後の日時が過去日時である | エラー表示、Step3 へ戻る |
| EXC-3 | Step2 | 対象が開始済みまたはキャンセル済み | 変更できない旨を表示し、Step1 へ戻る |

# 6. 事後条件

| No | 条件 |
|---|---|
| 1 | 予約の日時・タイトルが変更後の内容で登録されている |
| 2 | 変更前の内容は残っていない |
