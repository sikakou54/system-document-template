# 1. 基本情報

| 項目 | 内容 |
|---|---|
| UC ID | UC-006 |
| ユースケース名 | 利用実績を確認する |
| 主アクター | 管理者 |
| トレース元 | FR-006 |
| 概要 | 管理者が会議室ごとの月次利用実績を確認する |

# 2. 事前条件

| No | 条件 |
|---|---|
| 1 | 管理者としてログイン済み(UC-007 完了) |
| 2 | 集計対象となる予約の実績が蓄積されている |

# 3. 基本フロー

| Step | アクター | 操作/処理 | 結果 |
|---|---|---|---|
| 1 | 管理者 | 利用実績を開く | 利用実績画面が表示される |
| 2 | 管理者 | 対象月を指定する | 集計が要求される |
| 3 | システム | 指定月に利用が完了した予約を会議室ごとに集計する | 会議室別の利用件数・利用時間が算出される |
| 4 | システム | 会議室別の利用実績を提示する | 利用実績が表示される |

# 4. 代替フロー

| ALT ID | 分岐Step | 条件 | フロー |
|---|---|---|---|
| ALT-1 | Step4 | 対象月に完了した予約が0件 | 実績が0件であることを確認し、対象月を変えて再確認する |

# 5. 例外フロー

| EXC ID | 発生Step | 条件 | 対応 |
|---|---|---|---|
| - | - | なし | - |

# 6. 事後条件

| No | 条件 |
|---|---|
| 1 | 指定月の会議室別利用実績が管理者に提示されている |
| 2 | 予約データは変更されていない |
