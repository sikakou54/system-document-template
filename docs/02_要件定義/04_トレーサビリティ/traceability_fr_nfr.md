# 1. 概要

機能要件(FR)に適用される共通機能要件(CFR)と非機能要件(NFR)を追跡する。

- CFR-001〜005・CFR-007 は機能横断の共通機能要件(正本 CFR-000_システム共通要件.md／詳細は各 CFR-XXX_名称.md)。CFR-006(通知)は採番予約で未確認のため本表に現れない。CFR-008(共通ナビゲーション)はログイン後の機能導線であり個別 FR への適用ではないため本表の対象外。
- NFR-001〜006 は 02_要件定義/02_非機能要件/ に個別文書として定義する(NFR-001 性能=会議室検索固有、NFR-002〜005=機能横断、NFR-006 監査=重要操作を伴う機能)。
- 適用CFR は各機能の性質から整理する(認証=要ログインの全機能／認可=ロール・本人制御を伴う機能／入力検証=利用者入力を伴う機能／エラー表示=画面でエラーを扱う機能／一覧検索・ページング=一覧・検索を伴う機能)。

# 3. 更新ルール

| 更新契機 | 更新内容 |
|---|---|
| FR の新規作成・変更 | 該当 FR の行を追加／更新し、適用CFR・適用NFR を各文書から転記する |
| NFR(NFR-XXX)の追加・変更 | 全 FR 行の適用NFRを 02_要件定義/02_非機能要件/index.md の一覧に合わせて更新する |
| ID廃止 | 行は削除せず、備考に「廃止」と記す |

- 本対応表の更新は関連文書の作成・更新と同一作業内で行う(正本: docs/CLAUDE.md §6・§10-7)。
