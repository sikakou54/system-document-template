RQ は 01_要求定義/要求一覧.md、実現要件は 02_要件定義/01_機能要件/・02_要件定義/02_非機能要件/ の各文書からの転記で構成する。

# 3. 更新ルール

| 更新契機 | 更新内容 |
|---|---|
| RQ の新規作成・変更 | 該当 RQ の行を追加／更新し、実現要件を関連文書から転記する |
| FR/CFR/NFR の追加・変更 | 実現する RQ を確認し、実現要件列を更新する |
| ID廃止 | 行は削除せず、備考に「廃止」と記す |

- 本対応表の更新は関連文書の作成・更新と同一作業内で行う(正本: docs/CLAUDE.md §6・§10-7)。
