# 1. 概要

MeetRoom が取り扱う情報を、エンティティごとのデータ定義書として定義する。

- 各エンティティのデータ項目・ライフサイクル(CRUD)・関連機能/業務ルール・セキュリティを管理する。
- 全エンティティ間の関連は 概念ER図.md を参照。
- データの業務オーナー・正本(SoR)・書込責任・保持期間と状態遷移は データ所有・状態遷移.md を参照。
- 各定義は本ディレクトリの MDL-XXX_○○情報.md に記載する。
- 区分値・権限は要件レベルで自己完結的に記載し、物理型・物理名は記載しない（設計層IDは参照しない）。
