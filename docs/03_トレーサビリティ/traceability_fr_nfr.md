# traceability_fr_nfr

## 1. 概要

機能要件(FR)に適用される共通機能要件(CFR)・共通非機能要件(NFR)と、機能固有の非機能要件の有無を追跡する。
CFR-001〜005・NFR-002〜006 は機能横断の共通要件(正本 CFR-000_システム共通要件.md)。機能固有NFR は各 FR の「機能固有の非機能要件」節を参照する。
適用CFR は各機能の性質から整理する(認証=要ログインの全機能／認可=ロール・本人制御を伴う機能／入力検証=利用者入力を伴う機能／エラー表示=画面でエラーを扱う機能／一覧検索・ページング=一覧・検索を伴う機能)。

## 2. 対応表

| FR ID | 適用CFR | 適用NFR | 機能固有NFR |
|---|---|---|---|
| FR-001 会議室検索 | CFR-001, CFR-002, CFR-003, CFR-004, CFR-005 | NFR-002, NFR-003, NFR-004, NFR-005, NFR-006 | 有(NFR-001 性能) |
| FR-002 会議室予約 | CFR-001, CFR-002, CFR-003, CFR-004 | NFR-002, NFR-003, NFR-004, NFR-005, NFR-006 | 無 |
| FR-003 予約変更・キャンセル | CFR-001, CFR-002, CFR-003, CFR-004, CFR-005 | NFR-002, NFR-003, NFR-004, NFR-005, NFR-006 | 無 |
| FR-004 予約リマインド通知 | - | NFR-002, NFR-003, NFR-004, NFR-005, NFR-006 | 無 |
| FR-005 会議室・設備管理 | CFR-001, CFR-002, CFR-003, CFR-004 | NFR-002, NFR-003, NFR-004, NFR-005, NFR-006 | 無 |
| FR-006 利用実績レポート | CFR-001, CFR-002, CFR-004, CFR-005 | NFR-002, NFR-003, NFR-004, NFR-005, NFR-006 | 無 |
| FR-008 会議室利用の従量課金 | CFR-001, CFR-002, CFR-003, CFR-004 | NFR-002, NFR-003, NFR-004, NFR-005, NFR-006 | 無 |

- FR-004 は Cron/Queue 起動の背景処理(JOB-001)で実現し、画面・利用者入力を伴わないため共通機能要件(CFR-001〜005)は適用対象外(「-」)。
- NFR-001(性能)は FR-001 の機能固有NFRとして採番済みのため、共通NFR側では欠番(SPEC §4)。

## 3. 更新ルール

| 更新契機 | 更新内容 |
|---|---|
| FR の新規作成・変更 | 該当 FR の行を追加／更新し、適用CFR・適用NFR・機能固有NFRの有無を各文書から転記する |
| 共通NFR(NFR-XXX)の追加・変更 | 全 FR 行の適用NFRを CFR-000 の共通NFR一覧に合わせて更新する |
| ID廃止 | 行は削除せず、備考に「廃止」と記す |

- 本対応表の更新は関連文書の作成・更新と同一作業内で行う(正本: docs/CLAUDE.md §6・§10-7)。
