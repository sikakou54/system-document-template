# traceability_fr_nfr

# 1. 概要

機能要件(FR)に適用される共通機能要件(CFR)と非機能要件(NFR)を追跡する。
CFR-001〜005 は機能横断の共通機能要件(正本 CFR-000_システム共通要件.md／詳細は各 CFR-XXX_名称.md)。NFR-001〜006 は 02_要件定義/03_非機能要件/ に個別文書として定義する(NFR-001 性能=会議室検索固有、NFR-002〜005=機能横断、NFR-006 監査=重要操作を伴う機能)。
適用CFR は各機能の性質から整理する(認証=要ログインの全機能／認可=ロール・本人制御を伴う機能／入力検証=利用者入力を伴う機能／エラー表示=画面でエラーを扱う機能／一覧検索・ページング=一覧・検索を伴う機能)。

# 2. 対応表

| FR ID | 適用CFR | 適用NFR |
|---|---|---|
| FR-001 会議室検索 | CFR-001, CFR-002, CFR-003, CFR-004, CFR-005 | NFR-001, NFR-002, NFR-003, NFR-004, NFR-005 |
| FR-002 会議室予約 | CFR-001, CFR-002, CFR-003, CFR-004 | NFR-002, NFR-003, NFR-004, NFR-005, NFR-006 |
| FR-003 予約変更・キャンセル | CFR-001, CFR-002, CFR-003, CFR-004, CFR-005 | NFR-002, NFR-003, NFR-004, NFR-005, NFR-006 |
| FR-004 予約リマインド通知 | - | NFR-002, NFR-003, NFR-004, NFR-005 |
| FR-005 会議室・設備管理 | CFR-001, CFR-002, CFR-003, CFR-004 | NFR-002, NFR-003, NFR-004, NFR-005, NFR-006 |
| FR-006 利用実績レポート | CFR-001, CFR-002, CFR-004, CFR-005 | NFR-002, NFR-003, NFR-004, NFR-005 |
| FR-008 会議室利用の従量課金 | CFR-001, CFR-002, CFR-003, CFR-004 | NFR-002, NFR-003, NFR-004, NFR-005, NFR-006 |

- FR-004 は Cron/Queue 起動の背景処理(JOB-001)で実現し、画面・利用者入力を伴わないため共通機能要件(CFR-001〜005)は適用対象外(「-」)。
- NFR-001(性能)は会議室検索(FR-001)固有のため FR-001 にのみ適用する。NFR-002〜005(可用性・セキュリティ・通信暗号化・データ保護)は機能横断で全 FR に適用する。
- NFR-006(監査)は重要操作(予約・会議室管理・課金)を伴う FR-002・FR-003・FR-005・FR-008 に適用する。参照系の FR-001(検索)・FR-006(レポート)、通知の FR-004 は監査対象の重要操作を持たないため対象外。

# 3. 更新ルール

| 更新契機 | 更新内容 |
|---|---|
| FR の新規作成・変更 | 該当 FR の行を追加／更新し、適用CFR・適用NFR を各文書から転記する |
| NFR(NFR-XXX)の追加・変更 | 全 FR 行の適用NFRを 02_要件定義/03_非機能要件/index.md の一覧に合わせて更新する |
| ID廃止 | 行は削除せず、備考に「廃止」と記す |

- 本対応表の更新は関連文書の作成・更新と同一作業内で行う(正本: docs/CLAUDE.md §6・§10-7)。
