# 1. 基本情報

| 項目 | 内容 |
|---|---|
| クエリID | SQL-011 |
| クエリ名 | 会議室登録 |
| 概要 | 会議室を1件登録し、登録した会議室を返す。 |
| 用途 | 会議室の新規登録(RoomManagementService)。 |
| 使用元 | MOD-004 |

# 2. 対象テーブル

| テーブル | 物理名 | 用途 |
|---|---|---|
| TBL-002 | M_ROOMS | 登録対象(会議室の INSERT) |

# 3. 入力パラメータ

| パラメータ | 論理名 | 型 | 必須 | 説明・制約 |
|---|---|---|---|---|
| :name | 会議室名 | TEXT | Yes | 会議室名 |
| :capacity | 収容人数 | INTEGER | Yes | 1以上 |
| :location | 設置場所 | TEXT | Yes | 設置場所 |
| :hourly_rate | 利用単価 | INTEGER | Yes | 0以上(0=無料) |
| :status | 会議室ステータス | INTEGER | Yes | DEF-001/CODE-003。未指定時は既定値を設定(呼び出し側で既定化) |
| :note | 備考 | TEXT | No | 備考(NULL可) |

# 4. クエリ

会議室を INSERT する。CREATED_AT・UPDATED_AT は Default(CURRENT_TIMESTAMP)で設定される。登録した会議室を返す。

```sql
INSERT INTO M_ROOMS (
  NAME, CAPACITY, LOCATION, HOURLY_RATE, STATUS, NOTE
) VALUES (
  :name, :capacity, :location, :hourly_rate, :status, :note   -- STATUS: DEF-001/CODE-003
)
RETURNING ID, NAME, CAPACITY, LOCATION, HOURLY_RATE, STATUS, NOTE;
```

# 5. 出力(列)

| 列名 | 論理名 | 型 | 説明 |
|---|---|---|---|
| ID | 会議室ID | INTEGER | 採番された会議室の ID |
| NAME | 会議室名 | TEXT | 会議室名 |
| CAPACITY | 収容人数 | INTEGER | 収容人数 |
| LOCATION | 設置場所 | TEXT | 設置場所 |
| HOURLY_RATE | 利用単価 | INTEGER | 1時間あたり利用単価(円)。0=無料 |
| STATUS | 会議室ステータス | INTEGER | DEF-001/CODE-003 |
| NOTE | 備考 | TEXT | 備考(NULL可) |

# 6. 補足(性能・インデックス)

| 項目 | 内容 |
|---|---|
| 利用インデックス | なし(INSERT) |
| 想定件数 | 1件 |
| 注意点 | 入力値の妥当性(必須・型・収容人数・単価・ステータス許可値)は呼び出しモジュール(MOD-004)で検証済みとする。設備の紐付けは SQL-018 で別途行う |
