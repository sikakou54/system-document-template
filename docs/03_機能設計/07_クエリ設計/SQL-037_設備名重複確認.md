# 1. 基本情報

| 項目 | 内容 |
|---|---|
| クエリID | SQL-037 |
| クエリ名 | 設備名重複確認 |
| 概要 | 指定した設備名と同名の、有効な設備マスタの件数を返す。 |
| 用途 | 設備登録時の設備名重複確認(同名の設備が既に存在するかの検証。ERR-011)。 |
| 使用元 | MOD-004 |

# 2. 対象テーブル

| テーブル | 物理名 | 用途 |
|---|---|---|
| TBL-004 | M_EQUIPMENTS | 重複確認対象(設備名の実在判定) |

# 3. 入力パラメータ

| パラメータ | 論理名 | 型 | 必須 | 説明・制約 |
|---|---|---|---|---|
| :name | 設備名 | TEXT | Yes | 重複確認対象の設備名 |

# 4. クエリ

指定した設備名と同名の、有効(DELETED_AT IS NULL)な設備の件数を数える。呼び出し側で「件数 ＞ 0」なら同名設備が存在すると判定する。

```sql
SELECT
  COUNT(*) AS FOUND_COUNT
FROM M_EQUIPMENTS
WHERE NAME = :name
  AND DELETED_AT IS NULL;
```

# 5. 出力(列)

| 列名 | 論理名 | 型 | 説明 |
|---|---|---|---|
| FOUND_COUNT | 存在件数 | INTEGER | 指定した設備名と同名の有効な設備の件数 |

# 6. 補足(性能・インデックス)

| 項目 | 内容 |
|---|---|
| 利用インデックス | 設備名(NAME)での検索 |
| 想定件数 | 1件(件数を1行で返す) |
| 注意点 | FOUND_COUNT ＞ 0 なら同名設備が存在するため設備名重複(ERR-011)。DELETED_AT IS NULL の有効な設備のみを対象とする |
