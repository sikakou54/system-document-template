# 1. 基本情報

| 項目 | 内容 |
|---|---|
| クエリID | SQL-030 |
| クエリ名 | 月次利用実績取得 |
| 概要 | 対象月の会議室別利用実績を、会議室名付き・会議室名昇順で取得する。未集計の場合は 0件。 |
| 用途 | 利用実績レポートの取得(UsageReportService)。 |
| 使用元 | MOD-005 |

# 2. 対象テーブル

| テーブル | 物理名 | 用途 |
|---|---|---|
| TBL-006 | T_USAGE_REPORTS | 抽出対象(会議室別利用実績) |
| TBL-002 | M_ROOMS | 会議室名(NAME)の取得 |

# 3. 入力パラメータ

| パラメータ | 論理名 | 型 | 必須 | 説明・制約 |
|---|---|---|---|---|
| :target_month | 対象月 | TEXT | Yes | 'YYYY-MM' 形式 |

# 4. クエリ

対象月(TARGET_MONTH)一致かつ有効(DELETED_AT IS NULL)の利用実績を、ROOM_ID で M_ROOMS に結合して会議室名付きで取得する。会議室名昇順で並べる。

```sql
SELECT
  rep.ROOM_ID           AS ROOM_ID,
  m.NAME                AS ROOM_NAME,
  rep.TARGET_MONTH      AS TARGET_MONTH,
  rep.RESERVATION_COUNT AS RESERVATION_COUNT,
  rep.TOTAL_MINUTES     AS TOTAL_MINUTES
FROM T_USAGE_REPORTS rep
INNER JOIN M_ROOMS m
  ON m.ID = rep.ROOM_ID
WHERE rep.TARGET_MONTH = :target_month
  AND rep.DELETED_AT IS NULL
ORDER BY m.NAME ASC, rep.ROOM_ID ASC;
```

# 5. 出力(列)

| 列名 | 論理名 | 型 | 説明 |
|---|---|---|---|
| ROOM_ID | 会議室ID | INTEGER | 会議室の ID |
| ROOM_NAME | 会議室名 | TEXT | 会議室名 |
| TARGET_MONTH | 対象月 | TEXT | 対象の年月('YYYY-MM') |
| RESERVATION_COUNT | 予約件数 | INTEGER | 完了予約の件数 |
| TOTAL_MINUTES | 利用時間(分) | INTEGER | 利用時間の合計(分) |
