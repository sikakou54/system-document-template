# 1. 概要

全テーブルに共通で付与するカラムの正本。

- 各 TBL 文書には共通カラムを再記載しない(本ファイルのみで管理する)。
- 主キー ID は共通カラムに含めず、各 TBL 文書の §2 カラム定義に PK=◯ で記載する。
- DB基盤は Cloudflare D1(SQLite)。

# 2. 共通カラム一覧

| カラム名 | 型 | NULL | Key | Default | 説明 |
|---|---|---|---|---|---|
| CREATED_AT | TEXT | No | - | CURRENT_TIMESTAMP | 作成日時(ISO8601) |
| UPDATED_AT | TEXT | No | - | CURRENT_TIMESTAMP | 更新日時(ISO8601)。更新時にアプリケーションで現在時刻を設定する |
| DELETED_AT | TEXT | Yes | - | NULL | 論理削除。NULL=有効 / 値あり=削除済(削除日時) |
