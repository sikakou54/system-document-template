# 1. 概要

MeetRoom の機能設計層(画面 SCR・API・モジュール MOD・テーブル TBL など)内のトレースを一元管理する。
画面から API・モジュール・テーブルまでの実装連鎖など、機能設計内の要素間の対応を追跡する。工程をまたぐトレース(RQ→要件→設計)は最上位 04_トレーサビリティ/、要件定義層内の横断は 02_要件定義/04_トレーサビリティ/ で管理する。
