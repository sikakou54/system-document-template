# 1. 基本情報

| 項目 | 内容 |
|---|---|
| テーブルID | TBL-009 |
| 物理名 | T_AUDIT_LOGS |
| 論理名 | 監査ログトランザクション |
| 概要 | 予約・課金・会議室管理などの重要操作の実行時に、システムが操作の記録(監査証跡)を追記する監査ログトランザクション(CFR-007／NFR-006)。記録後は追記のみで更新・削除しない。 |
