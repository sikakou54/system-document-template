# 1. 概要

全テーブルに共通で付与するカラムの正本。

- 各 TBL 文書には共通カラムを再記載しない(本ファイルのみで管理する)。
- 主キー ID は共通カラムに含めず、各 TBL 文書の §2 カラム定義に PK=◯ で記載する。
- DB基盤は Cloudflare D1(SQLite)。

# 2. 共通カラム一覧

| カラム名 | 型 | NULL | Key | Default | 説明 |
|---|---|---|---|---|---|
| CREATED_AT | TEXT | No | - | `strftime('%Y-%m-%dT%H:%M:%fZ','now')` | 作成日時(RFC3339 UTC) |
| UPDATED_AT | TEXT | No | - | `strftime('%Y-%m-%dT%H:%M:%fZ','now')` | 更新日時(RFC3339 UTC)。更新時にアプリケーションで同形式の現在時刻を設定する |
| DELETED_AT | TEXT | Yes | - | NULL | 論理削除。NULL=有効 / 値あり=削除日時(RFC3339 UTC) |

- 日時TEXTの比較可能性を保つため、`YYYY-MM-DDTHH:mm:ss.SSSZ` のUTC固定形式だけを保存する。SQLiteの `CURRENT_TIMESTAMP` が返す空白区切り形式と混在させない。
