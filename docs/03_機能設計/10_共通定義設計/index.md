# 1. 概要

関連SEQと、テーブル設計・API設計・モジュール設計・JOB設計・画面設計・クエリ設計から共通参照される定義情報を管理する。
コード値や区分値の値・表示名・説明は本ディレクトリを正本とし、各文書では定義IDのみ参照する。

- 作成・更新前に `../11_トレーサビリティ/traceability_sequence_design.md` で関連SEQを確認する。
