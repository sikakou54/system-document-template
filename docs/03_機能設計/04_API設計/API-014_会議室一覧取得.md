# 1. 基本情報

| 項目 | 内容 |
|---|---|
| API ID | API-014 |
| API名 | 会議室一覧取得 |
| メソッド | GET |
| パス | /api/admin/rooms |
| 認証 | 要 |
| 認可 | 一般=不可, 管理者=可 |
| 冪等性 | あり(参照系) |
| トレース元 | FR-005/UC-01 |
| 概要 | 管理者向けに全会議室(利用停止を含む)を設備一覧付きで取得する。会議室・設備管理画面で編集対象を選択・現在値表示するために用いる。 |

# 2. リクエスト

| 項目名 | 型 | 必須 | 説明・制約 |
|---|---|---|---|
| なし | - | - | 入力項目なし(全会議室を返す) |

# 3. レスポンス

| 項目 | 内容 |
|---|---|
| HTTPステータス | 200 |

以下は items 配列の各要素。

| 項目名 | 型 | 説明 |
|---|---|---|
| 会議室ID | int | 会議室の一意な識別子 |
| 会議室名 | string | 会議室の名称 |
| 収容人数 | int | 収容できる人数 |
| 設置場所 | string | 会議室の場所 |
| 利用単価 | int | 1時間あたり利用単価(円)。0=無料 |
| 会議室ステータス | int | 共通コード定義/CODE-003(利用停止を含む) |
| 備考 | string | 会議室の備考 |
| 設備一覧 | array | 会議室に紐づく設備一覧。要素の構造は以下のとおり |
| 設備ID | int | 設備の一意な識別子 |
| 設備名 | string | 設備の名称 |

# 4. 処理フロー

この API の基本フローをフローチャートで定義する。

```mermaid
flowchart TD
  Start([開始])
  P1[["(1) 会議室一覧取得"]]
```

# 5. 処理詳細

処理フローの各処理で行う内容を定義する。

## (1) 会議室一覧取得

利用停止を含む全会議室を設備一覧付きで取得する。該当が無い場合は空一覧を返す。

| MOD-ID | 処理名 |
|---|---|
| MOD-004 | 会議室一覧取得処理 |

| 引数項目 | 値 |
|---|---|
| なし | - |

## (2) 処理結果

(1) 会議室一覧取得の結果をレスポンスとして返却する。

| 項目名 | データ型 | 値 | 説明 |

| 会議室一覧 | Object[] | (1) 会議室一覧取得の結果 | 返却する会議室一覧 |

| - 会議室名 | String | (1) 会議室一覧取得の結果 | 返却する会議室名 |
| - 収容人数 | Integer | (1) 会議室一覧取得の結果 | 返却する収容人数 |
| - 設置場所 | String | (1) 会議室一覧取得の結果 | 返却する設置場所 |
| - 利用単価 | Integer | (1) 会議室一覧取得の結果 | 返却する利用単価 |
| - 会議室ステータス | Integer | (1) 会議室一覧取得の結果 | 返却する会議室ステータス |
| - 備考 | String | (1) 会議室一覧取得の結果 | 返却する備考 |
| - 設備一覧 | Object[] | (1) 会議室一覧取得の結果 | 返却する設備一覧 |
| -- 設備ID | Integer | (1) 会議室一覧取得の結果 | 返却する設備ID |
