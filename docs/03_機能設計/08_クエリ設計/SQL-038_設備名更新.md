# 1. 基本情報

| 項目 | 内容 |
|---|---|
| クエリID | SQL-038 |
| クエリ名 | 設備名更新 |
| 概要 | 設備1件の設備名を更新し、更新後の設備を返す。 |
| 用途 | 設備名の編集(RoomManagementService)。 |
| 使用元 | MOD-004 |

# 2. 対象テーブル

| テーブル | 物理名 | 用途 |
|---|---|---|
| TBL-004 | M_EQUIPMENTS | 更新対象(設備名の UPDATE) |

# 3. 入力パラメータ

| パラメータ | 論理名 | 型 | 必須 | 説明・制約 |
|---|---|---|---|---|
| :id | 設備ID | INTEGER | Yes | 更新対象設備の ID |
| :name | 設備名 | TEXT | Yes | 更新後の設備名(50文字以内)。UX_EQUIPMENTS_NAME で一意 |
| :updated_at | 更新日時 | TEXT | Yes | ISO8601(UTC)。アプリで現在時刻を設定(共通カラム.md) |

# 4. クエリ

有効(DELETED_AT IS NULL)な設備の設備名を更新する。更新後の設備を返す。

```sql
UPDATE M_EQUIPMENTS
SET NAME       = :name,
    UPDATED_AT = :updated_at
WHERE ID = :id
  AND DELETED_AT IS NULL
RETURNING ID, NAME;
```

# 5. 出力(列)

| 列名 | 論理名 | 型 | 説明 |
|---|---|---|---|
| ID | 設備ID | INTEGER | 設備の ID |
| NAME | 設備名 | TEXT | 設備名 |

# 6. 補足(性能・インデックス)

| 項目 | 内容 |
|---|---|
| 利用インデックス | 主キー(ID)による一意更新。UX_EQUIPMENTS_NAME(NAME 一意制約)により同名は制約違反となる |
| 想定件数 | 最大1件 |
| 注意点 | 対象設備の存在確認(SQL-014)・同名重複確認(SQL-037)は呼び出しモジュール(MOD-004)で事前に行う。入力値の妥当性は呼び出しモジュール(MOD-004)で検証済みとする。UPDATED_AT はアプリで現在時刻を設定する |
