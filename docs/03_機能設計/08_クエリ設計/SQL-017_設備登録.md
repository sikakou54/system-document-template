# 1. 基本情報

| 項目 | 内容 |
|---|---|
| クエリID | SQL-017 |
| クエリ名 | 設備登録 |
| 概要 | 設備を1件登録し、登録した設備を返す。 |
| 用途 | 設備マスタの新規登録(RoomManagementService)。 |
| 使用元 | MOD-004 |

# 2. 対象テーブル

| テーブル | 物理名 | 用途 |
|---|---|---|
| TBL-004 | M_EQUIPMENTS | 登録対象(設備の INSERT) |

# 3. 入力パラメータ

| パラメータ | 論理名 | 型 | 必須 | 説明・制約 |
|---|---|---|---|---|
| :name | 設備名 | TEXT | Yes | 設備名(50文字以内)。UX_EQUIPMENTS_NAME で一意 |

# 4. クエリ

設備を INSERT する。登録した設備を返す。

```sql
INSERT INTO M_EQUIPMENTS (
  NAME
) VALUES (
  :name
)
RETURNING ID, NAME;
```

# 5. 出力(列)

| 列名 | 論理名 | 型 | 説明 |
|---|---|---|---|
| ID | 設備ID | INTEGER | 採番された設備の ID |
| NAME | 設備名 | TEXT | 設備名 |

# 6. 補足(性能・インデックス)

| 項目 | 内容 |
|---|---|
| 利用インデックス | UX_EQUIPMENTS_NAME(NAME 一意制約)。同名登録は制約違反となる |
| 想定件数 | 1件 |
| 注意点 | 同名重複は事前に SQL-015 で確認する(ERR-011)。入力値の妥当性は呼び出しモジュール(MOD-004)で検証済みとする |
