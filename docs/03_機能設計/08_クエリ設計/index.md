# 1. 概要

MeetRoom のクエリ設計(SQL)を定義する。空き会議室検索・月次利用実績集計など、モジュールから利用する SQLite(D1)クエリを管理する。各クエリの詳細は SQL-XXX_クエリ名.md に記載する。

- 作成・更新前に `../11_トレーサビリティ/シーケンス設計トレーサビリティ.md` で関連SEQを特定し、データ参照更新から目的・入力・出力・対象MDL/TBLを抽出する。
- SQLの実行元はMODだけとし、API・JOBから直接実行しない。
