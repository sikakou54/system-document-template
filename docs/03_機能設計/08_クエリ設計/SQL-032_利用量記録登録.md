# 1. 基本情報

| 項目 | 内容 |
|---|---|
| クエリID | SQL-032 |
| クエリ名 | 利用量記録登録 |
| 概要 | 利用量記録を1件登録し(計上ステータス未送信で作成)、登録した記録を返す。 |
| 用途 | 完了した有料予約の利用量記録登録(BillingService)。 |
| 使用元 | MOD-007 |

# 2. 対象テーブル

| テーブル | 物理名 | 用途 |
|---|---|---|
| TBL-007 | T_USAGE_RECORDS | 登録対象(利用量記録の INSERT) |

# 3. 入力パラメータ

| パラメータ | 論理名 | 型 | 必須 | 説明・制約 |
|---|---|---|---|---|
| :reservation_id | 予約ID | INTEGER | Yes | 計上対象の予約ID |
| :user_id | ユーザーID | INTEGER | Yes | 予約者の ID |
| :room_id | 会議室ID | INTEGER | Yes | 予約対象の会議室ID |
| :usage_minutes | 利用時間分 | INTEGER | Yes | 利用時間(分) |
| :unit_price | 適用単価 | INTEGER | Yes | 円/時。利用時点の会議室単価 |
| :amount | 金額 | INTEGER | Yes | 円(参考値)。USAGE_MINUTES ÷ 60 × UNIT_PRICE |

# 4. クエリ

利用量記録を INSERT する。DEF-001/SET-011で作成する。登録した記録を返す。

```sql
INSERT INTO T_USAGE_RECORDS (
  RESERVATION_ID, USER_ID, ROOM_ID, USAGE_MINUTES, UNIT_PRICE, AMOUNT, STATUS
) VALUES (
  :reservation_id, :user_id, :room_id, :usage_minutes, :unit_price, :amount, 1   -- DEF-001/SET-011
)
RETURNING ID, RESERVATION_ID, USER_ID, ROOM_ID, USAGE_MINUTES, UNIT_PRICE, AMOUNT, STATUS;
```

# 5. 出力(列)

| 列名 | 論理名 | 型 | 説明 |
|---|---|---|---|
| ID | 利用量記録ID | INTEGER | 採番された利用量記録の ID |
| RESERVATION_ID | 予約ID | INTEGER | 計上対象の予約ID |
| USER_ID | ユーザーID | INTEGER | 予約者の ID |
| ROOM_ID | 会議室ID | INTEGER | 会議室の ID |
| USAGE_MINUTES | 利用時間分 | INTEGER | 利用時間(分) |
| UNIT_PRICE | 適用単価 | INTEGER | 円/時。利用時点の会議室単価 |
| AMOUNT | 金額 | INTEGER | 円(参考値) |
| STATUS | 計上ステータス | INTEGER | DEF-001/CODE-006 |

# 6. 補足(性能・インデックス)

|---|---|
| 利用インデックス | UX_USAGE_RECORDS_RESERVATION(RESERVATION_ID 一意制約) |
| 想定件数 | 1件 |
| 注意点 | 二重計上は事前に SQL-031 で確認する(UX_USAGE_RECORDS_RESERVATION により予約ごと1件)。利用時間・金額は呼び出しモジュール(MOD-007)で算出する。送信後の状態更新は SQL-033 |
