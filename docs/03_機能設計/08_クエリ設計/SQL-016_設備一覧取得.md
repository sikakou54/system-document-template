# 1. 基本情報

| 項目 | 内容 |
|---|---|
| クエリID | SQL-016 |
| クエリ名 | 設備一覧取得 |
| 概要 | 有効な設備マスタを設備名昇順で全件取得する。 |
| 用途 | 設備マスタの一覧取得(会議室検索の設備条件、会議室登録編集の設備選択)。 |
| 使用元 | MOD-004 |

# 2. 対象テーブル

| テーブル | 物理名 | 用途 |
|---|---|---|
| TBL-004 | M_EQUIPMENTS | 抽出対象(設備マスタの全件) |

# 3. 入力パラメータ

| パラメータ | 論理名 | 型 | 必須 | 説明・制約 |
|---|---|---|---|---|
| なし | - | - | - | 引数なし(設備マスタ全件が対象) |

# 4. クエリ

有効(DELETED_AT IS NULL)な設備を設備名昇順で全件取得する。

```sql
SELECT
  e.ID   AS ID,
  e.NAME AS NAME
FROM M_EQUIPMENTS e
WHERE e.DELETED_AT IS NULL
ORDER BY e.NAME ASC, e.ID ASC;
```

# 5. 出力(列)

| 列名 | 論理名 | 型 | 説明 |
|---|---|---|---|
| ID | 設備ID | INTEGER | 設備の ID |
| NAME | 設備名 | TEXT | 設備名 |

# 6. 補足(性能・インデックス)

| 項目 | 内容 |
|---|---|
| 利用インデックス | 全件走査(設備は少数)。設備名順は TBL-004/IDX-1(UX_EQUIPMENTS_NAME)が寄与 |
| 想定件数 | 設備マスタの全件(少数) |
| 注意点 | ページングは行わず全件返す |
