# 1. 基本情報

| 項目 | 内容 |
|---|---|
| クエリID | SQL-021 |
| クエリ名 | 予約取得 |
| 概要 | 予約ID一致かつ論理削除されていない有効な予約を1件取得する。該当が無い場合は 0件。 |
| 用途 | 利用量計上時の予約取得(利用時間算出・課金対象の特定)。 |
| 使用元 | MOD-007 |

# 2. 対象テーブル

| テーブル | 物理名 | 用途 |
|---|---|---|
| TBL-003 | T_RESERVATIONS | 抽出対象(IDによる予約取得) |

# 3. 入力パラメータ

| パラメータ | 論理名 | 型 | 必須 | 説明・制約 |
|---|---|---|---|---|
| :id | 予約ID | INTEGER | Yes | 取得対象の予約 ID(主キー) |

# 4. クエリ

予約ID(ID)一致かつ有効(DELETED_AT IS NULL)の予約を1件取得する。利用時間算出に用いる USER_ID・ROOM_ID・START_AT・END_AT を返す。

```sql
SELECT
  r.ID       AS ID,
  r.ROOM_ID  AS ROOM_ID,
  r.USER_ID  AS USER_ID,
  r.TITLE    AS TITLE,
  r.START_AT AS START_AT,
  r.END_AT   AS END_AT,
  r.STATUS   AS STATUS
FROM T_RESERVATIONS r
WHERE r.ID = :id
  AND r.DELETED_AT IS NULL
LIMIT 1;
```

# 5. 出力(列)

| 列名 | 論理名 | 型 | 説明 |
|---|---|---|---|
| ID | 予約ID | INTEGER | 予約の ID |
| ROOM_ID | 会議室ID | INTEGER | 予約対象の会議室ID |
| USER_ID | ユーザーID | INTEGER | 予約者の ID |
| TITLE | 予約タイトル | TEXT | 予約タイトル |
| START_AT | 利用開始日時 | TEXT | ISO8601(UTC) |
| END_AT | 利用終了日時 | TEXT | ISO8601(UTC) |
| STATUS | 予約ステータス | INTEGER | DEF-001/CODE-004 |

# 6. 補足(性能・インデックス)

| 項目 | 内容 |
|---|---|
| 利用インデックス | 主キー(ID)による一意検索 |
| 想定件数 | 最大1件 |
| 注意点 | 予約者に限定しない(ID のみで取得)。利用時間(分)は START_AT・END_AT から呼び出しモジュール側で算出する |
