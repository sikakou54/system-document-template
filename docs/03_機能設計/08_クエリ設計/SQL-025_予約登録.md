# 1. 基本情報

| 項目 | 内容 |
|---|---|
| クエリID | SQL-025 |
| クエリ名 | 予約登録 |
| 概要 | 利用可能会議室かつ重複予約なしを同一SQL文で再確認し、予約時単価をスナップショットして1件登録する。条件不成立は0件を返す。 |
| 用途 | 予約の登録(ReservationService)。 |
| 使用元 | MOD-003 |

# 2. 対象テーブル

| テーブル | 物理名 | 用途 |
|---|---|---|
| TBL-003 | T_RESERVATIONS | 登録対象(予約の INSERT) |
| TBL-002 | M_ROOMS | 利用可否と予約確定時単価の参照 |

# 3. 入力パラメータ

| パラメータ | 論理名 | 型 | 必須 | 説明・制約 |
|---|---|---|---|---|
| :room_id | 会議室ID | INTEGER | Yes | 予約対象の会議室ID |
| :user_id | ユーザーID | INTEGER | Yes | 予約する利用者の ID |
| :title | 予約タイトル | TEXT | Yes | 予約タイトル |
| :start_at | 利用開始日時 | TEXT | Yes | ISO8601(UTC)。:start_at ＜ :end_at |
| :end_at | 利用終了日時 | TEXT | Yes | ISO8601(UTC) |
| :now | 判定基準時刻 | TEXT | Yes | RFC3339 UTC。利用開始日時がこの時刻以降であること |

# 4. クエリ

会議室の利用可能性、過去日時でないこと、重複予約がないことをINSERT文内で評価する。会議室の現在単価をUNIT_PRICEへ保存する。

```sql
INSERT INTO T_RESERVATIONS (
  ROOM_ID, USER_ID, TITLE, START_AT, END_AT, UNIT_PRICE, STATUS, REMIND_STATUS
)
SELECT
  r.ID, :user_id, :title, :start_at, :end_at, r.HOURLY_RATE, 1, 1
FROM M_ROOMS r
WHERE r.ID = :room_id
  AND r.STATUS = 1
  AND r.DELETED_AT IS NULL
  AND :start_at >= :now
  AND :start_at < :end_at
  AND NOT EXISTS (
    SELECT 1
    FROM T_RESERVATIONS x
    WHERE x.ROOM_ID = r.ID
      AND x.STATUS = 1
      AND x.DELETED_AT IS NULL
      AND x.START_AT < :end_at
      AND x.END_AT > :start_at
  )
RETURNING ID, ROOM_ID, USER_ID, TITLE, START_AT, END_AT, UNIT_PRICE, STATUS, REMIND_STATUS;
```

# 5. 出力(列)

| 列名 | 論理名 | 型 | 説明 |
|---|---|---|---|
| ID | 予約ID | INTEGER | 採番された予約の ID |
| ROOM_ID | 会議室ID | INTEGER | 予約対象の会議室ID |
| USER_ID | ユーザーID | INTEGER | 予約者の ID |
| TITLE | 予約タイトル | TEXT | 予約タイトル |
| START_AT | 利用開始日時 | TEXT | ISO8601(UTC) |
| END_AT | 利用終了日時 | TEXT | ISO8601(UTC) |
| UNIT_PRICE | 予約確定時単価 | INTEGER | 会議室から取得して予約へ保存した単価 |
| STATUS | 予約ステータス | INTEGER | 共通コード定義/CODE-004 |
| REMIND_STATUS | リマインド通知ステータス | INTEGER | 共通コード定義/CODE-005 |
