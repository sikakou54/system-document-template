# 1. 基本情報

| 項目 | 内容 |
|---|---|
| クエリID | SQL-013 |
| クエリ名 | 会議室設備名一覧取得 |
| 概要 | 指定した会議室が備える設備名の一覧を、設備名昇順で取得する。該当が無い場合は 0件。 |
| 用途 | 会議室詳細(getRoomById)の設備一覧取得。 |
| 使用元 | MOD-002 |

# 2. 対象テーブル

| テーブル | 物理名 | 用途 |
|---|---|---|
| TBL-005 | M_ROOM_EQUIPMENTS | 会議室と設備の紐付け(絞り込み) |
| TBL-004 | M_EQUIPMENTS | 設備名の取得 |

# 3. 入力パラメータ

| パラメータ | 論理名 | 型 | 必須 | 説明・制約 |
|---|---|---|---|---|
| :room_id | 会議室ID | INTEGER | Yes | 対象会議室の ID |

# 4. クエリ

指定会議室(ROOM_ID)の紐付け(M_ROOM_EQUIPMENTS)を EQUIPMENT_ID で M_EQUIPMENTS に結合し、有効(DELETED_AT IS NULL)な設備名を設備名昇順で取得する。

```sql
SELECT
  e.ID   AS EQUIPMENT_ID,
  e.NAME AS EQUIPMENT_NAME
FROM M_ROOM_EQUIPMENTS re
INNER JOIN M_EQUIPMENTS e
  ON e.ID = re.EQUIPMENT_ID
  AND e.DELETED_AT IS NULL
WHERE re.ROOM_ID = :room_id
  AND re.DELETED_AT IS NULL
ORDER BY e.NAME ASC;
```

# 5. 出力(列)

| 列名 | 論理名 | 型 | 説明 |
|---|---|---|---|
| EQUIPMENT_ID | 設備ID | INTEGER | 設備の ID |
| EQUIPMENT_NAME | 設備名 | TEXT | 設備名 |
