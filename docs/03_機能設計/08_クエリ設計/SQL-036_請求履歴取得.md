# 1. 基本情報

| 項目 | 内容 |
|---|---|
| クエリID | SQL-036 |
| クエリ名 | 請求履歴取得 |
| 概要 | 利用者の請求履歴を、請求対象月の降順で取得する。 |
| 用途 | 課金利用状況の請求履歴取得(BillingService)。 |
| 使用元 | MOD-007 |

# 2. 対象テーブル

| テーブル | 物理名 | 用途 |
|---|---|---|
| TBL-008 | T_INVOICES | 抽出対象(利用者の請求履歴) |

# 3. 入力パラメータ

| パラメータ | 論理名 | 型 | 必須 | 説明・制約 |
|---|---|---|---|---|
| :user_id | ユーザーID | INTEGER | Yes | 請求履歴を取得する利用者の ID |

# 4. クエリ

利用者(USER_ID)一致かつ有効(DELETED_AT IS NULL)の請求を、請求対象月降順で取得する。

```sql
SELECT
  i.ID                AS ID,
  i.USER_ID           AS USER_ID,
  i.STRIPE_INVOICE_ID AS STRIPE_INVOICE_ID,
  i.BILLING_PERIOD    AS BILLING_PERIOD,
  i.AMOUNT            AS AMOUNT,
  i.STATUS            AS STATUS
FROM T_INVOICES i
WHERE i.USER_ID = :user_id
  AND i.DELETED_AT IS NULL
ORDER BY i.BILLING_PERIOD DESC, i.ID DESC;
```

# 5. 出力(列)

| 列名 | 論理名 | 型 | 説明 |
|---|---|---|---|
| ID | 請求ID | INTEGER | 請求行の ID |
| USER_ID | ユーザーID | INTEGER | 利用者の ID |
| STRIPE_INVOICE_ID | Stripe請求ID | TEXT | Stripe の invoice ID |
| BILLING_PERIOD | 請求対象月 | TEXT | 'YYYY-MM' 形式 |
| AMOUNT | 請求金額 | INTEGER | 円 |
| STATUS | 請求ステータス | INTEGER | DEF-001/CODE-007 |

# 6. 補足(性能・インデックス)

| 項目 | 内容 |
|---|---|
| 利用インデックス | USER_ID による絞り込み |
| 想定件数 | 利用者ごとの請求件数(月次) |
| 注意点 | 並び順は請求対象月降順(BILLING_PERIOD DESC, ID DESC)。ページング(page / limit)と総件数は API-COM §5 に従い呼び出し側で適用・算出する |
