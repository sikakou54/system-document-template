# 1. 基本情報

| 項目 | 内容 |
|---|---|
| クエリID | SQL-015 |
| クエリ名 | 設備取得 |
| 概要 | 設備名一致かつ論理削除されていない有効な設備を1件取得する。該当が無い場合は 0件。 |
| 用途 | 設備登録時の同名重複確認(UX_EQUIPMENTS_NAME 違反の事前判定)。 |
| 使用元 | MOD-004 |

# 2. 対象テーブル

| テーブル | 物理名 | 用途 |
|---|---|---|
| TBL-004 | M_EQUIPMENTS | 抽出対象(設備名による設備取得) |

# 3. 入力パラメータ

| パラメータ | 論理名 | 型 | 必須 | 説明・制約 |
|---|---|---|---|---|
| :name | 設備名 | TEXT | Yes | 重複確認する設備名。UX_EQUIPMENTS_NAME |

# 4. クエリ

設備名(NAME)一致かつ有効(DELETED_AT IS NULL)の設備を1件取得する。取得できれば同名設備が既に存在すると判定する。

```sql
SELECT
  e.ID   AS ID,
  e.NAME AS NAME
FROM M_EQUIPMENTS e
WHERE e.NAME = :name
  AND e.DELETED_AT IS NULL
LIMIT 1;
```

# 5. 出力(列)

| 列名 | 論理名 | 型 | 説明 |
|---|---|---|---|
| ID | 設備ID | INTEGER | 設備の ID |
| NAME | 設備名 | TEXT | 設備名 |

# 6. 補足(性能・インデックス)

| 項目 | 内容 |
|---|---|
| 利用インデックス | TBL-004/IDX-1(UX_EQUIPMENTS_NAME: NAME) |
| 想定件数 | 最大1件(NAME は一意) |
| 注意点 | 取得できた場合は同名重複(ERR-011 相当)と判定する。判定は呼び出しモジュール(MOD-004)で行う |
